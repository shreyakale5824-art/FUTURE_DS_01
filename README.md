# Sales Data Analysis Dashboard

This project analyzes a retail sales dataset using Microsoft Excel. 
Pivot tables and charts were used to explore sales performance across regions, product categories, and monthly trends. 
A dashboard was created to visualize key insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Dashboard Features
- Sales by Region
- Monthly Sales Trend
- Sales by Category
- Top Products by Sales

## Key Insights
- West region has the highest sales performance.
- Technology category generates the most revenue.
- Sales increase toward the end of the year.
- A few products contribute most of the revenue.
