# Data Science & Analytics Internship - Future Interns

This repository contains the tasks completed as part of the Data Science & Analytics Internship at Future Interns. The projects focus on data analysis, visualization, and deriving actionable insights using real-world datasets.

---

## 📊 Task 1: Sales Data Analysis Dashboard

### 🔹 Objective

Analyze sales data to identify trends, regional performance, and product-level insights.

### 🔹 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization

### 🔹 Key Analysis

* Sales by Region
* Monthly Sales Trends
* Sales by Category
* Top Products by Sales

### 🔹 Key Insights

* West region generated the highest sales
* Technology category contributed the most revenue
* Sales increased toward the end of the year
* A few products contributed significantly to total revenue

## 📉 Task 2: Customer Churn & Retention Analysis

### 🔹 Objective

Analyze customer data to understand churn behavior, retention patterns, and customer lifetime trends.

### 🔹 Tools Used

* Python (Jupyter Notebook)
* Pandas, Matplotlib, Seaborn

### 🔹 Key Analysis

* Churn rate calculation
* Retention analysis using tenure
* Monthly charges vs churn
* Customer lifetime trends

### 🔹 Key Insights

* Customers with higher monthly charges are more likely to churn
* Customers with low tenure show higher churn rates
* Long-term customers have better retention
* Early-stage customer engagement is critical for retention

### 🔹 Recommendations

* Improve onboarding experience
* Offer incentives for long-term subscriptions
* Focus on high-risk customers to reduce churn

## 📈 Task 3: Marketing Funnel & Conversion Performance Analysis

### 🔹 Objective

Analyze marketing funnel data to identify conversion drop-offs, channel performance, and opportunities to improve lead-to-customer conversion.

### 🔹 Tools Used

* Microsoft Excel / Python
* Data Visualization

### 🔹 Key Analysis

* Funnel stage analysis (Leads → Qualified → Converted)
* Conversion rate at each stage
* Drop-off identification between funnel stages
* Channel performance comparison

### 🔹 Key Insights

* Significant drop-off observed between lead generation and conversion stages
* Certain marketing channels show higher conversion efficiency
* Early funnel stages attract more users, but fewer progress to final conversion
* Conversion rates vary across different acquisition sources

### 🔹 Recommendations

* Optimize landing pages to improve initial engagement
* Improve targeting strategies for high-quality leads
* Focus on high-performing marketing channels
* Enhance follow-up strategies to reduce drop-offs
